# Environment Variables

## Required

- `APP_NAME`
- `APP_ENV`
- `APP_KEY`
- `APP_URL`
- `DB_CONNECTION`
- `DB_DATABASE` / `DB_HOST` / `DB_PORT` / `DB_USERNAME` / `DB_PASSWORD`

## Recommended

- `QUEUE_CONNECTION=database`
- `SESSION_DRIVER=database`
- `CACHE_STORE=database`

## TrackEd-specific

- `AI_PRECHECKER_URL`
- `AI_PRECHECKER_TOKEN`
- `AI_PRECHECKER_TIMEOUT`
- `SEEDED_USER_PASSWORD`
