# Runbook

## Common Operator Tasks

### New account provisioning
1. Sign in as admin
2. Open Personnel
3. Create user and assign PBAC permissions
4. Share the temporary password securely

### LIS sync
1. Open LIS Sync
2. Upload the latest CSV export
3. Verify inserted / updated / skipped counts

### Blocked clearance investigation
1. Open Admin Clearances
2. Identify unresolved damaged/lost assets
3. Update assignment or asset condition once settled

### Good moral issuance
1. Open Good Moral
2. Search by student LRN
3. Confirm no unresolved major incidents
4. Download certificate
